ReST Tryouts
============

Inline code with blanks at the beginning
========================================

Using U+0020 SPACE
------------------

- ``:literal:`\ abc``` → :literal:`\ abc`
- ```\ abc``` → `\ abc`
- \`\` abc\`\` → ``\ abc``

Using U+00A0 NO-BREAK SPACE
---------------------------

- ``:literal:``` abc` → :literal:` abc`
- ``` abc``` → ` abc`
- '' abc'' → `` abc``
